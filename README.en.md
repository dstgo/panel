## Introduction

Wendy Panel is a distributed Don't Starve Together management panel for managing dedicated server clusters of Don't Starve Together. It is based on the modern front-end project template [vben](https://github.com/vbenjs/vue-vben-admin), which is implemented by Vue3, Vite5, ant-design-vue, Pinia, UnoCSS and Typescript. Wendy Panel can quickly build a Don't Starve server management platform and maintain and manage it very conveniently.
